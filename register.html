রেজিস্টার করুন

ভুবনডাঙ্গার কবিতায় লেখক/পাঠক হিসেবে যুক্ত হতে আপনার basic তথ্য দিন।

Profile photo optional। ছবি দিলে approval-এর পরে public profile-এ দেখা যাবে।
আপনার registration request গ্রহণ করা হয়েছে।
Supabase চালু হলে account তৈরি হয়ে profile page-এ আপনার তথ্য দেখা যাবে।

Profile/Bio/Photo update approval-এর পরে public হবে।
এখন এই registration form static demo। পরে Supabase Auth চালু হলে email/password দিয়ে real account তৈরি হবে।
আগেই account আছে? লগইন করুন

Registration-এর পরে কী হবে?

  • আপনার profile তৈরি হবে।
  • আপনি নিজের profile ও লেখক পরিচিতি edit করতে পারবেন।
  • Edit করলে পরিবর্তন সরাসরি public হবে না; approval লাগবে।
  • আপনি কবিতা, প্রলাপ ও অনুকাব্য জমা দিতে পারবেন।
  • নতুন লেখা approval-এর পরে public হবে।
  • অনুকাব্য public feed-এ নয়; profile-এ থাকবে।

Daily Limit

সাধারণ user প্রতিদিন সর্বোচ্চ ২টি কবিতা, ২টি প্রলাপ এবং ৫টি অনুকাব্য জমা দিতে পারবে। Founder/Admin-এর জন্য limit থাকবে না।
