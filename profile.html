মানবিক আবেদন ও Homepage Pin Request

জরুরি মানবিক আবেদন আলাদা Form থেকে পাঠানো হবে। কবিতা, প্রলাপ, চিঠি, আবৃত্তি বা পরিবেশ Submission-এর মধ্যে মানবিক আবেদন রাখা হবে না।

আপনার মানবিক আবেদনের অবস্থা

বর্তমানে কোনো Pin Request পাঠানো হয়নি।

কোনো Request নেই

প্রথমে মানবিক আবেদন Form পূরণ করুন। আবেদন তৈরি হওয়ার পর এখান থেকে Homepage-এ সাময়িকভাবে Pin করার Review Request পাঠানো যাবে।

মানবিক আবেদন Form
আপনার Pin Request গ্রহণ করা হয়েছে।
এটি এখনো Public Homepage-এ Pin হয়নি। আপনার বার্তাটি যাচাই করে Admin Panel থেকে ভুবনডাঙ্গা Team অনুমোদন করলে সক্রিয় জরুরি প্রয়োজনের জন্য সেটি সাময়িকভাবে Homepage-এর উপরের অংশে Pin করা হবে। Backend যুক্ত হলে এই Request-এর Notification Admin Panel-এ পৌঁছাবে।
বর্তমান Static সীমাবদ্ধতা: এই Page-এ Request পাঠালে শুধু আপনার Browser-এ Pending অবস্থা দেখা যাবে। কোনো বাস্তব Notification এখন Admin Panel বা Website Team-এর কাছে পাঠানো হবে না। Real Notification, Verification, Approval এবং Public Pin চালু করতে Backend ও Database প্রয়োজন।
