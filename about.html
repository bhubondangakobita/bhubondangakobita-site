ভুবনডাঙ্গার কবিতা

বাংলা সাহিত্য ও সৃজনশীলতার একটি মুক্ত ভুবন

হোম খ্যাতিমান কবি আজকের কবিতা আপনার প্রোফাইল মধ্যরাতের প্রলাপ মাহমুদ সোহেল সাহিত্য আলোচনা
🌿

আমাদের সম্পর্কে / নীতিমালা

ভুবনডাঙ্গার কবিতা বাংলা সাহিত্য, কবিতা, প্রলাপ, অনুকাব্য, আবৃত্তি ও সাহিত্য আলোচনার জন্য একটি সম্মানজনক সৃজনশীল ভুবন।

ভুবনডাঙ্গার কবিতা কী?

ভুবনডাঙ্গার কবিতা এমন একটি সাহিত্যভুবন, যেখানে নতুন ও পুরোনো লেখক, পাঠক, কবিতাপ্রেমী এবং সৃজনশীল মানুষ নিজেদের ভাবনা প্রকাশের জায়গা পাবে।

এখানে কবিতা শুধু শব্দের সৌন্দর্য নয়; এখানে কবিতা মানুষের ভেতরের সত্য, একাকীত্ব, ভালোবাসা, আত্মসম্মান, অভিমান, ঈশ্বর, সমাজ এবং জীবনের প্রশ্নের ভাষা।

“নিজের প্রতিও শ্রদ্ধা রেখো, ভালোবাসা রেখো মনে। কেউ মুগ্ধ হবেনা জেনেও যেমন— ফুল ফোটে বনে।”

_ মাহমুদ সোহেল

কেন এই platform?

অনেক মানুষ লিখতে চান, কিন্তু সমাজের ভয়, বিচার, উপহাস, সংকোচ বা আত্মবিশ্বাসের অভাবে নিজের লেখা প্রকাশ করতে পারেন না। ভুবনডাঙ্গার কবিতা সেই নীরব সৃষ্টিশীল মানুষদের জন্য তৈরি।

কবিতা

প্রেম, ব্যথা, নিঃসঙ্গতা, জীবন, আত্মসম্মান, প্রকৃতি ও মানুষের ভেতরের কথা।

প্রলাপ

ভেতরের ভাবনা, একান্ত কথোপকথন, রাত, ঈশ্বর, সমাজ ও নিজের সঙ্গে নিজের কথা।

অনুকাব্য

ছোটো পঙক্তি, ক্ষুদ্র ভাবনা, সংক্ষিপ্ত কাব্য—যা শুধু লেখকের profile-এ থাকবে।

সাহিত্য আলোচনা

কবিতা, বই, ভাষা, আবৃত্তি, সাহিত্য ভাবনা ও সৃজনশীল আলোচনা।

লেখা প্রকাশের নিয়ম

  • অন্যের লেখা নিজের নামে প্রকাশ করা যাবে না।
  • অশালীন, ঘৃণামূলক, ব্যক্তিগত আক্রমণাত্মক বা ক্ষতিকর লেখা গ্রহণযোগ্য নয়।
  • নতুন লেখা submit করলে সরাসরি public হবে না; approval-এর পরে publish হবে।
  • লেখা edit করলেও সেটি pending approval-এ যাবে।
  • Admin যাচাই করে approve/reject করার অধিকার রাখবে।
  • Copyright issue থাকলে যোগাযোগ / কপিরাইট ইস্যু page ব্যবহার করতে হবে।
ভুবনডাঙ্গা সাহিত্য টিমের উদ্দেশ্য কোনো লেখককে অপমান করা নয়; বরং platform-টি সুন্দর, নিরাপদ ও সম্মানজনক রাখা।

Profile Edit এবং Pending Approval

প্রত্যেক user নিজের profile, লেখক পরিচিতি, profile photo এবং নিজের লেখা edit করতে পারবে। তবে edit করলেই পরিবর্তন সরাসরি public হবে না।

  • User নিজের profile edit করলে সেটি pending review-এ যাবে।
  • লেখক পরিচিতি edit করলেও approval লাগবে।
  • Profile photo edit করলেও approval লাগবে।
  • কবিতা, প্রলাপ বা অনুকাব্য edit করলেও pending approval হবে।
  • Admin approve করলে public page-এ update দেখা যাবে।
  • Reject হলে আগের approved version থাকবে।
এই নিয়ম platform-এর নিরাপত্তা, মান এবং লেখকের মর্যাদা রক্ষার জন্য।

Copyright Policy

  • Copyright অভিযোগের জন্য লেখার title/link দিতে হবে।
  • মূল লেখক বা source সম্পর্কে তথ্য দিতে হবে।
  • প্রমাণ থাকলে তা উল্লেখ করতে হবে।
  • অভিযোগ যাচাই না করে কোনো content remove করা হবে না।
  • প্রয়োজনে content temporarily hidden করা যেতে পারে।

খ্যাতিমান কবি section

খ্যাতিমান কবিদের page-এ পূর্ণ কবিতা না দিয়ে পরিচিতি এবং নির্বাচিত রচনার নাম রাখা হবে। কারণ অনেক আধুনিক ও সমসাময়িক কবির লেখা copyright protected হতে পারে।

Public domain বা অনুমতিপ্রাপ্ত লেখা হলে ভবিষ্যতে পূর্ণ কবিতা যুক্ত করা যাবে।

লেখা কপি প্রতিরোধ

Website-এ casual copy কমানোর জন্য text selection, right click এবং copy command নিরুৎসাহিত/বন্ধ রাখা হয়েছে। তবে internet-এ কোনো লেখা ১০০% copy-proof করা যায় না।

  • Right click বন্ধ থাকবে।
  • Text select বন্ধ থাকবে।
  • Copy command বন্ধ থাকবে।
  • Copy/right click করলে warning message দেখা যাবে।
  • Supabase চালু হলে original submission time, edit history এবং approval record থাকবে।
এতে লেখকের ownership proof রাখা সহজ হবে।

ভুবনডাঙ্গা সাহিত্য টিম

Review, copyright issue, report, feedback এবং content approval ধীরে ধীরে admin panel-এর মাধ্যমে পরিচালিত হবে। এখন website static হলেও ভবিষ্যতে Supabase যুক্ত হলে সবকিছু database-based হবে।

Review

নতুন লেখা, edit request এবং profile update review করা হবে।

Report

অশালীন লেখা, spam, copy বা ভুল তথ্য report করা যাবে।

Approval

Admin approve করলে লেখা বা edit public হবে।
