আপনার প্রকাশিত লেখায় মন্তব্যের অনুমতি

কে Mention করতে পারবে

Post বা Comment-এ আপনার নাম যুক্ত করার অনুমতি

ব্যক্তিগত Settings সম্পর্কে

Backend চালু হলে Privacy Settings Database-এ নিরাপদভাবে সংরক্ষিত হবে। সাধারণ অবস্থায় Founder বা Admin আপনার ব্যক্তিগত Privacy Settings, Password অথবা Private Draft দেখতে পারবেন না।

Block ও Restriction

অপ্রয়োজনীয় যোগাযোগ ও হয়রানি নিয়ন্ত্রণ করুন

Comment ও Interaction Restriction

আপনার Profile-এ অপ্রীতিকর কার্যক্রম সীমিত করুন

অপরিচিত Account-এর Comment Review করুন

নতুন বা Follow না করা Account-এর Comment সরাসরি দেখা যাবে না

অশালীন শব্দ Filter করুন

সম্ভাব্য অশালীন Comment Review তালিকায় যাবে

Repeated Message Request সীমিত করুন

একই Account থেকে বারবার Message Request এলে সীমিত হবে

Account নিরাপত্তা

Password, Login ও ব্যক্তিগত Message সম্পর্কিত সুরক্ষা

Password পরিবর্তন

শক্তিশালী Password ব্যবহার করুন। Backend চালু হলে Password নিরাপদ Hash হিসেবে সংরক্ষিত হবে; Founder বা Admin মূল Password দেখতে পারবেন না।

Two-Step Verification

ভবিষ্যতে Email বা Authenticator-এর মাধ্যমে অতিরিক্ত Login Verification চালু করা হবে।

Login Session

Backend চালু হলে কোন Device ও Browser থেকে Account ব্যবহার করা হচ্ছে, তা User নিজে দেখতে এবং Session বন্ধ করতে পারবেন।

ব্যক্তিগত Message-এর গোপনীয়তা

ব্যক্তিগত Message সাধারণ অবস্থায় Founder বা Admin পড়তে পারবেন না। কোনো ব্যবহারকারী অশ্লীলতা, হয়রানি, হুমকি বা প্রতারণার অভিযোগ করলে Report করা নির্দিষ্ট Message অংশ Moderation Review-এর জন্য জমা দেওয়া যেতে পারে।

প্রমাণ ছাড়া শাস্তি নয়

শুধু অভিযোগের ভিত্তিতে Ban করা হবে না। Report করা তথ্য, Message, প্রসঙ্গ এবং প্রমাণ পর্যালোচনার পর Warning, সাময়িক Restriction অথবা স্থায়ী Ban দেওয়া হবে।

Account নিয়ন্ত্রণ

Data export, Account pause ও Account removal

আপনার তথ্যের কপি

Backend চালু হলে নিজের Profile তথ্য, লেখা ও সাহিত্যিক অবদানের একটি কপি Download করার অনুরোধ করা যাবে।

Account সাময়িকভাবে বন্ধ করুন

Deactivate করলে Profile সাধারণ ব্যবহারকারীদের কাছে গোপন থাকবে। পরে Login করে পুনরায় চালু করা যাবে।

Account মুছে ফেলার অনুরোধ

স্থায়ীভাবে Account মুছে ফেলার আগে পরিচয় যাচাই ও নির্ধারিত অপেক্ষার সময় থাকবে। আইনগত বা Moderation Record প্রয়োজন হলে সীমিত তথ্য নির্দিষ্ট সময় সংরক্ষিত থাকতে পারে।

বর্তমান Static সংস্করণে পরিবর্তনগুলো শুধু এই Browser-এ সংরক্ষিত হবে।

বাতিল

প্রাইভেসির মূলনীতি

আপনার নিয়ন্ত্রণ

Profile, লেখা, প্রিয় তালিকা ও সাহিত্যিক অবদান আলাদাভাবে Hide করা যাবে।

Founder-এর সীমিত ক্ষমতা

Founder সাধারণভাবে Password, Private Draft, ব্যক্তিগত Settings বা Message দেখতে পারবেন না।

Moderation প্রয়োজনভিত্তিক

Report, নিরাপত্তা বা নীতিমালা প্রয়োগের জন্য প্রয়োজনীয় তথ্যই পর্যালোচনা করা হবে।

Message নিরাপত্তা

সাধারণ Message ব্যক্তিগত

ব্যক্তিগত কথোপকথন সাধারণভাবে Platform কর্তৃপক্ষ পড়বে না।

Report করা অংশ

অভিযোগকারী Report করা নির্দিষ্ট Message প্রমাণ হিসেবে Moderation-এর কাছে পাঠাতে পারবেন।

প্রমাণিত অপরাধ

অশ্লীলতা, হয়রানি, হুমকি বা প্রতারণা প্রমাণিত হলে Restriction বা Ban হতে পারে।
