সাধারণ user শুধু নিজের profile ও নিজের লেখা manage করতে পারবে। Admin সব pending request approve/reject করতে পারবে।

Admin Rules

  • নতুন লেখা approve না করলে public হবে না।
  • Profile edit approve না করলে public profile update হবে না।
  • লেখক পরিচিতি edit approve না হলে আগের version থাকবে।
  • Copyright issue যাচাই না করে final decision নেওয়া উচিত নয়।
  • অশালীন বা harmful content temporarily hide করা যাবে।
  • Founder/Admin-এর posting limit থাকবে না।
  • সাধারণ user-এর daily limit system database থেকে control হবে।
